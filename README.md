# 💻 Modern Full-Stack Developer Portfolio

Welcome to my professional portfolio! This project serves as a central hub for my work, 
technical skills, and journey as a developer at the **National Open University of Nigeria**.

### ✨ Key Features:
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using Tailwind CSS.
- **Project Showcases:** Deep dives into my **College School Management System** (React/Node.js) and **Boutique E-commerce** (Paystack Integration).
- **Type Safety:** Built entirely with **TypeScript** for robust, maintainable code.
- **CI/CD Pipeline:** Automated deployments to GitHub Pages using GitHub Actions.
- **PWA Ready:** Installable on mobile devices for a native-app feel.
